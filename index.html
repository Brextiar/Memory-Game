ENI - MEMORY - Accueil

Bienvenue sur le site du Memory de l'ENI.

Venez jouer avec nous et faites le meilleur score possible.

Règles du jeu :

  • Retournez deux carte. Si les images sont identiques, le jeu laisse les cartes visibles.
  • Si les images sont différentes, les cartes se retournent faces cachées là où elles étaient.
  • La partie est terminée lorsque toutes les cartes ont étées assemblées par paires.

Si vous vouslez vous souvenir de votre score, inscrivez-vous via le formulaire.

Le jeux garde les 5 meilleurs scores.

Si vous voulez recommencer une partie appuyez sur la barre d'espace.
